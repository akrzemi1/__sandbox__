Concept Calculator

A Calculator is something that can be set a limit and then can be used to obtain a result of a computation.

Notation

TA type that is a model of Calculator
xAn lvalue of type T
iAn rvalue of type int

Valid expressions and types

T::result_typea type
T::set_limit(i)return type unimportant
x.get_result()returns const result_type&, throws nothing
